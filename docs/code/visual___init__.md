# `visual/__init__.py`

## Что это за файл

Это ещё один пустой технический файл.

## Зачем он нужен

Он помечает папку `visual` как Python-пакет.

## Почему это полезно

Даже если файл пустой, он помогает:

- поддерживать аккуратную структуру проекта,
- избегать проблем с импортами,
- держать единый стиль с папкой `src`.

## Простая мысль

Этот файл ничего не рисует и ничего не считает.
Он нужен только для того, чтобы остальной код было проще подключать и запускать.
