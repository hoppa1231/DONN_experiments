# Русский разбор кода

В этой папке лежит отдельный разбор каждого Python-файла проекта.

Идея простая:

- один файл кода,
- один отдельный markdown-файл с объяснением,
- всё на русском,
- без предположения, что читатель уже хорошо знает TensorFlow, DONN или саму статью.

## Файлы из `src/`

- `src___init__.md`
- `src_HopfLayer.md`
- `src_classifier.md`
- `src_demodulation.md`
- `src_operators.md`
- `src_sentiment.md`

## Файлы из `visual/`

- `visual___init__.md`
- `visual_classifier_result.md`
- `visual_demodulation_result.md`
- `visual_operators_result.md`
- `visual_sentiment_result.md`
- `visual_sentiment_paper_result.md`

## Как этим пользоваться

Если хочется понять проект по шагам, лучше идти в таком порядке:

1. `src_HopfLayer.md`
2. `src_classifier.md`
3. `visual_classifier_result.md`
4. `src_demodulation.md`
5. `visual_demodulation_result.md`
6. `src_operators.md`
7. `visual_operators_result.md`
8. `src_sentiment.md`
9. `visual_sentiment_result.md`
10. `visual_sentiment_paper_result.md`

Так чтение будет идти от общей DONN-механики к конкретным экспериментам.
