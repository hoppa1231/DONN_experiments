# `src/__init__.py`

## Что это за файл

Это технический файл-маркер.

Сам по себе он почти ничего не делает, потому что пустой.

## Зачем он нужен

Он говорит Python:

- папка `src` является пакетом,
- из неё можно стабильно импортировать код как из единого модуля.

## Что бы было без него

Без такого файла импорты вроде:

```python
from src.HopfLayer import HopfLayer
```

могут работать нестабильно в зависимости от того, откуда именно запущен скрипт.

## Почему файл всё равно важен

Новичкам часто кажется, что пустой файл не нужен.
Но в Python такие файлы часто играют чисто инфраструктурную роль:

- они не делают вычислений,
- но помогают всей структуре проекта работать предсказуемо.
